### 👨‍💻 About Me
I'm a Software Engineering student at HUFLIT, focused on Android development.  
I build real-world applications using Kotlin and Jetpack Compose, following modern practices like MVVM, Coroutines, and offline-first architecture.

Currently working on **SpendWise** — an offline-first expense manager with cloud sync using Firebase.

I care about understanding *why things work*, not just making them work.
